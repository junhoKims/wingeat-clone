## 💻 실행 방법

**원격 저장소 복제**

```bash
git clone https://github.com/tacotico/wingeat-clone
```

**프로젝트 폴더로 이동**

```bash
cd wingeat-clone
```

**패키지 설치**

```bash
npm install
```

**개발 서버 실행**

```bash
start on localhost:3000
npm start
```

**빌드**

```bash
npm run build
```
